# Kabataan Homepage Module Rebuild
**Date:** March 28, 2026

## Overview
Rebuilt the Homepage module (`app/Modules/Homepage`) to use the same visual style as the Dashboard feed. The page now serves as a public-facing landing page ‚Äî visitors can browse community posts without logging in, but must login or sign up to interact (like, comment, apply for programs).

---

## Changes Made

### 1. Homepage View ‚Äî `app/Modules/Homepage/Views/index.blade.php`
Completely rebuilt from the old prototype layout to match the Dashboard card/feed style.

**Key changes:**
- Replaced the old hero panel + 3-column layout with the Dashboard's navbar + feed + sidebar grid
- Navbar now shows **Login** and **Sign Up** buttons instead of a user profile/avatar menu
- Feed uses the same `post-card` structure as the Dashboard
- Filter tabs added: **All / Announcements / Events / Activities / Programs**
- Search bar filters posts by text in real time
- Right sidebar shows **Programs by Category** (10 categories with icons) and a **Santa Cruz Snapshot** highlights block
- Three login-gate modals added:
  - **Login Required** ‚Äî triggered when clicking any program category in the sidebar
  - **Action Login** ‚Äî triggered when clicking Like or Comment on any post
  - **Program Apply Login** ‚Äî triggered when clicking "View Program Details & Apply" on a Program-type post card

### 2. Homepage CSS ‚Äî `app/Modules/Homepage/assets/css/homepage.css`
Fully rewritten to match the Dashboard's design system.

**Includes:**
- Sticky navbar with SK OnePortal blue gradient
- `nav-auth-btn` styles (ghost outline for Login, white solid for Sign Up)
- Feed filter tab styles with active gradient state
- Post card styles (same as Dashboard: `post-card`, `post-type` badges, `post-actions`)
- Image placeholder with dashed border pattern
- Program category sidebar with hover color-swap effect
- Highlights sidebar card
- Modal styles (overlay, slide-in animation, close button)
- Responsive breakpoints: 1100px, 768px, 480px

### 3. Homepage JS ‚Äî `app/Modules/Homepage/assets/js/homepage.js`
Fully rewritten.

**Features:**
- Feed filter tabs ‚Äî filters `post-card` elements by `data-post-type`
- Search input ‚Äî filters posts by full text content
- Empty state shown when no posts match
- Program category click ‚Üí opens `loginRequiredModal` with category name injected
- Program apply button click ‚Üí opens `programApplyModal` with program name injected
- Like/Comment button click ‚Üí opens `actionLoginModal`
- ESC key closes any open modal
- Overlay click closes modals

### 4. Controller ‚Äî `app/Modules/Homepage/Controllers/HomepageController.php`
Updated feed data and view variables.

**Changes:**
- Removed `$barangays` and `$categories` variables (no longer used in the new layout)
- Added `$programCategories` array ‚Äî 10 categories each with `key`, `label`, `count`, and inline SVG `icon`
- Updated `$feedItems` to include all four post types:
  - 2 √ó **Event**
  - 2 √ó **Program** (Sports Development, Livelihood)
  - 2 √ó **Announcement**
  - 2 √ó **Activity** (Environment, Health)
- Updated `return view()` to pass `programCategories` instead of `barangays`/`categories`
