## 6. Setup Instructions

### 6.1 Installation Steps
```bash
# Navigate to Kabataan folder
cd Kabataan

# Install PHP dependencies
composer install

# Install Node dependencies
npm install

# Setup environment
copy .env.example .env
php artisan key:generate

# Configure database in .env file
# Run migrations
php artisan migrate

# Build assets
npm run build

# Start server
php artisan serve
```

### 6.2 Database Configuration
**Recommended:** Use Supabase PostgreSQL (same as Admin and SK_Federations)

```env
DB_CONNECTION=pgsql
DB_HOST=aws-1-ap-southeast-1.pooler.supabase.com
DB_PORT=6543
DB_DATABASE=postgres
DB_USERNAME=postgres.bxnsievezpzbevobukjg
DB_PASSWORD=SK_OnePortal67
DB_SSLMODE=require
DB_POOL_MODE=transaction
```

### 6.3 Session Configuration
```env
SESSION_DRIVER=database
SESSION_LIFETIME=45
SESSION_ENCRYPT=false
SESSION_PATH=/
SESSION_DOMAIN=null
SESSION_EXPIRE_ON_CLOSE=true
SESSION_SAME_SITE=strict
```

### 6.4 Clear Caches (Important!)
Before running the application, clear all Laravel caches:
```bash
php artisan config:clear
php artisan route:clear
php artisan view:clear
php artisan cache:clear
```

### 6.5 Verify Routes
Check if routes are properly registered:
```bash
php artisan route:list
```

You should see routes like:
- GET /login
- POST /login
- GET /dashboard
- POST /logout

- **URL:** http://localhost:8000/ (auto-redirects to login)
- **Direct Login URL:** http://localhost:8000/login
- **Development Mode:**

## 10. Multi-Step Registration Form

### 10.1 Design Specifications
- **Form Type:** 3-Step Multi-Step Form
- **Layout:** Side-by-side (Logo left, Form right) - same as login page
- **Design:** Modern, clean, organized, youth-friendly
- **Validation:** Real-time client-side validation with inline errors
- **Responsive:** Fully responsive with mobile stacking

### 10.2 Step Progress Indicator
**Features:**
- Visual step indicator with 3 circles (1, 2, 3)
- Active step highlighted in green
- Completed steps marked with green checkmark
- Progress lines between steps
- Step labels: "Personal & Contact", "Address & Verification", "Account Security"
- Responsive: Labels hidden on mobile (<640px)

### 10.3 Step 1: Personal & Contact Information

**Fields:**
1. **First Name** (Required)
   - Text input
   - Placeholder: "Juan"
   - Validation: Required

2. **Last Name** (Required)
   - Text input
   - Placeholder: "Dela Cruz"
   - Validation: Required

3. **Middle Initial** (Optional)
   - Text input, max 1 character
   - Placeholder: "M"
   - Marked as "(Optional)"

4. **Suffix** (Optional)
   - Dropdown select
   - Options: None, Jr., Sr., II, III, IV
   - Marked as "(Optional)"

5. **Birthdate** (Required)
   - Date input
   - Validation: Required, Age 15-30 years old
   - Auto-calculates age

6. **Age** (Auto-calculated, Read-only)
   - Text input (readonly)
   - Automatically populated when birthdate is selected
   - Background: Gray (disabled style)
   - Cursor: not-allowed
   - Shows validation message:
     - Green "✓ Eligible age" if 15-30 years old
     - Red "⚠️ Must be 15-30 years old" if outside range

7. **Email Address** (Required)
   - Email input
   - Placeholder: "juan@example.com"
   - Validation: Required, Valid email format

8. **Contact Number** (Optional)
   - Tel input, 11 digits
   - Placeholder: "09XX XXX XXXX"
   - Pattern: Must start with 09, exactly 11 digits
   - Auto-formats: Removes non-digits
   - Marked as "(Optional)"

**Navigation:**
- Next button: "Next: Address" (Green gradient button with arrow icon)

### 10.4 Step 2: Address & Verification

**Fields:**
1. **Province** (Read-only)
   - Text input
   - Value: "Laguna" (fixed)
   - Read-only field

2. **Municipality** (Read-only)
   - Text input
   - Value: "Santa Cruz" (fixed)
   - Read-only field

3. **Barangay** (Required)
   - Dropdown select
   - 26 barangays of Santa Cruz, Laguna:
     - Alipit, Bagumbayan, Bubukal, Calios, Duhat, Gatid, Jasaan, Labuin, Malinao, Oogong, Pagsawitan, Palasan, Patimbao
     - Poblacion I, II, III, IV, V
     - San Jose, San Juan, San Pablo Norte, San Pablo Sur
     - Santisima Cruz, Santo Angel Central, Santo Angel Norte, Santo Angel Sur
   - Validation: Required

4. **Upload Valid ID** (Required)
   - File input
   - Accepted types: School ID, Barangay ID, National ID
   - Accepted formats: JPG, PNG, PDF
   - Max size: 5MB
   - Drag and drop support
   - Shows file name and size after upload
   - Validation: Required, Max 5MB

**Navigation:**
- Back button: "Back" (Gray secondary button with left arrow)
- Next button: "Next: Account Security" (Green gradient button with arrow icon)

### 10.5 Step 3: Account Security & Agreement

**Fields:**
1. **Password** (Required)
   - Password input
   - Placeholder: "Minimum 8 characters"
   - Password toggle (eye icon)
   - Requirements hint: "Must contain: 8+ characters, 1 uppercase letter, 1 number"
   - Real-time strength indicator:
     - Weak (red): Less than requirements
     - Medium (yellow): Meets basic requirements
     - Strong (green): Exceeds requirements
   - Validation: Min 8 chars, 1 uppercase, 1 number

2. **Confirm Password** (Required)
   - Password input
   - Placeholder: "Re-enter your password"
   - Password toggle (eye icon)
   - Validation: Must match password

3. **Privacy Policy Agreement** (Required)
   - Checkbox
   - Text: "I agree to the Privacy Policy"
   - Privacy Policy link (clickable)
   - Validation: Must be checked

**Navigation:**
- Back button: "Back" (Gray secondary button with left arrow)
- Submit button: "Create Account" (Green gradient button with user-plus icon)
